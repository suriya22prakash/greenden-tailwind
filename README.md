# 🌿 Greenden – Plant Selling Website

Greenden is a responsive plant-selling website built using **HTML5** and **Tailwind CSS**. The project focuses on creating a clean, modern, and user-friendly interface while applying responsive design principles and Tailwind's utility-first approach. It was developed as a frontend practice project to strengthen my web development skills.

## 🚀 Features

* Responsive design for mobile, tablet, and desktop
* Modern landing page with a hero section
* Product showcase page
* Contact page
* Responsive navigation menu
* Clean and minimal user interface
* Utility-first styling with Tailwind CSS

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS
* Google Fonts
